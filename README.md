# Python_web
Python web server where you can successfully login
## Instalation
Program uses flask, flask_sqlalchemy and pyjwt.

```shell
-pip install Flask
-pip install Flask-SQLAlchemy
-pip install PyJWT
```

## Usage

To run type these commands:

```shell
cd src/
python3 python.py
```
